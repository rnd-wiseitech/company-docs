# 디자인 시스템

위세아이텍 제안서·PPT 공식 디자인 시스템 문서입니다.  
상세 명세는 [design.md](./design.md)를 참조하세요.

---

## 핵심 요약

### 테마
한국 공공·기업 제안서 양식 기반. **격식·구조·정보 밀도** 중심.  
인쇄된 제안서처럼 상단→하단 순서로 읽히며, 모든 콘텐츠 영역은 **빨간 점선 프레임**으로 구획됩니다.

---

### 주요 컴포넌트

- **슬라이드 제목 밴드**: 로마 숫자 탭 + Navy 하단 구분선
- **중간바 (KEY 바)**: Sky 400 배경 + 좌측 Navy "KEY" 필
- **테이블**: Navy 헤더 + 지브라 스트라이프 (White/Sky 100)
- **콘텐츠 프레임**: 페이지 내측 1px 점선 Red — **디자인 시그니처, 절대 제거 금지**
- **푸터**: 챕터명 (좌) + 로마숫자-페이지번호 (중앙) + WISEITECH (우)

---

### 캔버스 & 레이아웃

- **크기**: 19.1cm × 27.5cm (세로형, 인쇄 기반)
- **여백**: 외측 1.4cm + 콘텐츠 프레임 추가 0.45cm 인셋
- **정렬**: 한글 좌정렬, 제목·챕터 로마 숫자만 중앙정렬

---

### PPT 레이아웃 타입

| 타입 | 사용 조건 |
|------|----------|
| `standard` | 일반 서술형 |
| `table` | 목록·사양·비교 정보 |
| `metrics` | 성능·SLA·수치 지표 |
| `process` | 수행 절차·단계별 계획 |
| `arch` | 시스템 아키텍처·레이어 |
| `split` | 정량 데이터 + 설명 병렬 |

### 절대 금지 사항

- ❌ 그림자·글로우·글래스모피즘
- ❌ 16:9 와이드스크린 (명시적 요청 없으면 금지)
- ❌ 세리프 폰트
- ❌ 빨간 점선 프레임을 실선으로 교체
- ❌ 슬라이드당 KEY 바 2개 이상
- ❌ 한글 자간 조정

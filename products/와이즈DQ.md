# 와이즈DQ (WiseDQ)

> **분류:** 데이터 품질 관리 솔루션  
> **최종 수정일:** 2026-05-20

---

## 1. 제품 개요

AI 기술 기반으로 품질지표 설정·측정·결과 분석·모니터링을 수행하여  
**빅데이터 품질을 체계적으로 향상**시키는 데이터 품질 관리 플랫폼.

---

## 2. 주요 기능

| 기능 | 설명 |
|------|------|
| 품질지표 설정 | 데이터 특성에 맞는 품질 기준 및 측정 지표 정의 |
| 품질 진단 | 설정된 규칙 기반 데이터 품질 자동 측정·검증 |
| 결과 분석 | 품질 측정 결과 시각화 및 이슈 데이터 상세 분석 |
| 모니터링 | 데이터 품질 현황 실시간 모니터링 및 알림 |
| 품질 개선 | 오류 데이터 추적·수정 이력 관리 |

### 데이터버틀러 AI 기능 (DQ 영역)

| AI 기능 | 설명 | 구현 방향 |
|---------|------|-----------|
| 검증룰 자동 매핑 | DB 카탈로그·컬럼 메타데이터 기반으로 컬럼 특성에 적합한 검증룰 후보 자동 추천 | 룰 기반 + ML 기반 |
| 업무규칙 후보 자동 생성 | 컬럼 정보 기반으로 품질 진단에 활용 가능한 업무규칙 후보 자동 생성·추천 | ML 기반 |
| 자동 제외 룰 생성 | 컬럼 정보 기준으로 테이블·컬럼 진단 제외 대상 후보 자동 선별 | 룰 기반 |
| 간편 진단 | 검증룰 자동 매핑·업무규칙 추천·자동 제외 선별 통합 — 스키마 선택만으로 초기 진단 환경 구성 | 룰 기반 + ML 기반 연계 |

---

## 3. 도입 효과

- 품질 진단 준비 과정의 수작업 제거, 초기 셋업 시간 단축
- 업무규칙 작성 부담 감소, 진단 기준 수립 지원
- 불필요한 진단 대상 제거로 진단 효율성 향상
- 일관된 진단 기준 적용으로 품질 관리 표준화

## 5. 제안서 작성 시 강조 포인트

- **AI 자동화로 진단 셋업 시간 대폭 단축** — 기존 수작업 대비 효율화 수치 제시
- **데이터버틀러 AI 기능 탑재** — 검증룰 자동 매핑, 간편 진단으로 즉시 운영 가능
- **빅데이터 규모 대응** — 대용량 데이터 환경에서의 안정적 품질 진단
- **공공 데이터 품질 관리법 대응** — 법적 요구사항 충족을 위한 체계적 관리 근거

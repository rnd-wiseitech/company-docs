# ASTA

> **분류:** 시스템 영향도 분석 솔루션  
> **최종 수정일:** 2026-05-20

---

## 1. 제품 개요

시스템 영향도를 사전에 분석하여 **시스템 장애를 예방하고 변경 리스크를 최소화**하는  
IT 시스템 변경 관리 솔루션.

---

## 2. 주요 기능

| 기능 | 설명 |
|------|------|
| 영향도 분석 | 시스템 변경 시 연관 프로그램·DB·서비스 영향 범위 자동 분석 |
| 변경 관리 | 소스코드·쿼리 변경 이력 추적 및 관리 |
| 장애 예방 | 변경 전 영향도 사전 파악으로 예기치 않은 장애 방지 |
| 리스크 시각화 | 시스템 연관 구조 시각적 표현 |

---

## 3. 제안서 작성 시 강조 포인트

- **시스템 안정성 확보** — 변경 리스크 사전 파악으로 서비스 중단 방지
- **노후 시스템 현대화** — 레거시 시스템 구조 파악 후 단계적 전환 지원
- **Code Insure 연계** — 변경 영향도 분석 + 보안 취약점 점검 통합 관리

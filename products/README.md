---

## 작성 규칙

- 파일명: 제품명을 한글로 (예: `AI분석플랫폼.md`, `제안서자동화.md`)
- 인증·스펙 변경 시 해당 파일과 루트 `company_profile.json`을 함께 업데이트
- 도입 사례는 고객사 동의 후 기재

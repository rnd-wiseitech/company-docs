## 데이터버틀러 (Data Butler) AI 기능

**데이터버틀러**는 위세아이텍 AI 기능 전체를 통칭하는 브랜딩 명칭입니다.

---

## 작성 규칙

- 파일명: 제품명을 한글로 (예: `와이즈인텔리전스.md`, `와이즈DQ.md`)
- 인증·스펙 변경 시 해당 파일과 루트 `company_profile.json`을 함께 업데이트
- 각 파일 하단 **"제안서 작성 시 강조 포인트"** 섹션이 LLM에 우선 반영됨
- 새 제품 추가 시 `_양식.md` 복사 후 이 README 목록에도 추가

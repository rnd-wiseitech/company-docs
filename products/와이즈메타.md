# 와이즈메타 (WiseMeta)

> **분류:** 메타데이터 관리 솔루션  
> **최종 수정일:** 2026-05-20

---

## 1. 제품 개요

메타데이터 정보를 일관성 있게 정리하여 **데이터 이해도와 업무 생산성을 향상**시키는  
엔터프라이즈 메타데이터 관리 플랫폼.

---

## 2. 주요 기능

| 기능 | 설명 |
|------|------|
| 메타데이터 수집 | DB 카탈로그, 시스템 메타데이터 자동 수집 |
| 표준용어 관리 | 논리명·물리명 표준 체계 정의 및 관리 |
| 데이터 사전 | 용어·도메인·코드 체계 표준화 관리 |
| 데이터 계보 | 데이터 흐름·영향도 추적 (ASTA 연계) |
| 메타 검색 | 메타데이터 통합 검색 및 조회 |

### 데이터버틀러 AI 기능 (META 영역)

| AI 기능 | 설명 | 구현 방향 |
|---------|------|-----------|
| 표준용어 추천 기능 고도화 | 입력된 논리명에 부합하는 표준 물리명을 자동 추천 | 룰 기반 + 의미 기반 + ML 기반 (컬럼명 패턴 분석, 도메인 유형 분류) |

---

## 3. 도입 효과

- 표준 물리명 추천으로 사용자 작업 부담 감소, 작업 시간 단축
- 일관된 메타데이터 표준 체계 유지
- 데이터 거버넌스 기반 마련

---

## 4. 제안서 작성 시 강조 포인트

- **AI 기반 표준용어 자동 추천** — 메타데이터 구축 인력·시간 절감
- **데이터 거버넌스 체계 수립** — 공공기관 데이터 표준화 의무 대응
- **와이즈DQ와 연계** — 메타데이터 기반 품질 진단 체계 통합 운영
- **데이터버틀러 AI 기능** — 지속 고도화 중인 ML 기반 추천 엔진

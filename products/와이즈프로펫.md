# 와이즈프로펫 (WiseProphet)

> **분류:** 저코드 AI 개발 플랫폼  
> **최종 수정일:** 2026-05-20

---

## 1. 제품 개요

비전문가도 별도의 코딩 과정 없이 AI 모델을 개발하고 현업에 적용할 수 있는  
**저코드(Low-code) AI 개발 플랫폼**.

---

## 2. 주요 기능

| 기능 | 설명 |
|------|------|
| 노코드 모델 개발 | 드래그·드롭 방식으로 AI 모델 개발 파이프라인 구성 |
| 자동 머신러닝 | AutoML로 최적 알고리즘 자동 선택·하이퍼파라미터 튜닝 |
| 결과 시각화 | 모델 성능 및 예측 결과 직관적 시각화 |
| 현업 적용 | 개발된 모델을 업무 시스템에 API로 즉시 연동 |

---

## 3. 도입 효과

- AI 전문 인력 없이도 현업 담당자가 직접 AI 모델 개발·활용
- 모델 개발 기간 대폭 단축
- 데이터 기반 의사결정 문화 조직 전반 확산

---

## 4. 제안서 작성 시 강조 포인트

- **현업 중심 AI 민주화** — IT 부서 의존 없이 업무 담당자가 직접 AI 구현
- **WiseAIOps 연계** — 전문가·비전문가 모두를 위한 통합 AI 개발 환경
- **빠른 PoC** — 코딩 없이 수일 내 AI 모델 프로토타입 구현 가능

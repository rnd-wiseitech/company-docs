# 코드인슈어 (Code Insure)

> **분류:** 소스코드 보안 취약점 분석 솔루션  
> **최종 수정일:** 2026-05-20

---

## 1. 제품 개요

소스코드 보안 취약점을 사전에 점검·분석하여 **소프트웨어 보안성과 안정성을 확보**하는  
정적 분석(SAST) 기반 보안 솔루션.

---

## 2. 주요 기능

| 기능 | 설명 |
|------|------|
| 취약점 탐지 | 소스코드 정적 분석으로 보안 취약점 자동 탐지 |
| 취약점 분류 | CWE·OWASP 기준 취약점 유형별 분류·리포트 |
| 개선 가이드 | 취약점별 구체적 수정 가이드 제공 |
| CI/CD 연동 | 개발 파이프라인에 자동 보안 점검 통합 |

---

## 3. 제안서 작성 시 강조 포인트

- **개발 단계 보안 내재화(Shift Left)** — 운영 후 발견 대비 비용·리스크 대폭 절감
- **행안부 SW 보안 취약점 진단 요구사항 대응** — 공공기관 의무 대응
- **ASTA 연계** — 변경 영향도 분석 + 보안 취약점 점검 One-stop

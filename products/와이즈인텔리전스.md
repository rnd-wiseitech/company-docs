# 와이즈인텔리전스 (WiseIntelligence)

> **분류:** 빅데이터 분석·시각화 (BI 솔루션)  
> **최종 수정일:** 2026-05-20

---

## 1. 제품 개요

정형·비정형 데이터를 다양한 관점으로 분석하고 핵심 지표를 대시보드로 구성하여  
비즈니스 인사이트 도출과 효과적인 의사결정을 지원하는 **빅데이터 다차원 시각화 도구**.

- 100% HTML5 웹 기반 (반응형, Java 기반 커스터마이징 용이)
- 공공 및 금융 약 **200여 개 시스템** 적용
- 클라우드(SaaS)와 구축형(On-premise) 모두 지원
- 국내 최초 BI 솔루션 **GS인증 1등급** 획득
- SAP·MicroStrategy 등 외산 솔루션 **Win-back** 사례 다수

---

## 2. 주요 기능

| 기능 | 설명 |
|------|------|
| 다차원 비정형 분석 | Drag & Drop으로 분석 관점·항목 자유 선택. Drill Up/Down, Pivoting, Filtering, Sorting, Ranking 등 지원 |
| 정형 리포트 | 사용자 친숙한 Web엑셀 형태 보고서. 기존 업무 양식 그대로 활용 |
| 생성형 보고서 | AI 기반 정기 보고서 자동 생성·구축. 업무 보고서 활용한 정형 보고서 자동화 |
| 대시보드 | 다양한 레이아웃 재배치, 테마 스킨, 정형·비정형 보고서 공유 |
| 모니터링 | 카드·Map 형태 현황 가시화, 지표별 Alert, 목표 대비 실적(달성율) 관리 |
| 시각화 | 40여 종 이상 HTML5 차트 (막대, 파이, 트리맵, 히트맵, 지도매핑 등) |
| Self BI | 사용자가 파일(txt, csv, xls, xlsx 등) 직접 업로드 후 개인 보고서 작성 |
| 고급 분석 | 회귀분석·상관분석 등 20여 개 이상 고급통계 분석, 빅데이터 연동 |
| 보안 관리 | SHA-256 암호화, 사용자별 권한 관리, 접근 이력 조회, 개인정보보호법 준수 |
| 성능 모니터링 | 실시간 CPU·Memory·Disk 사용량 모니터링, 진행 중 작업 취소 |

---

## 3. 특장점

- **손쉬운 원클릭 분석**: 코딩 없이 Drag & Drop으로 즉시 시각화
- **40여 종 시각화 차트**: Tree-Map, Calendar View, Sankey Diagram 등 D3.js 확장
- **생성형 보고서 자동화**: LLM 기반 정기 보고서 자동 생성, 업무 효율 극대화
- **Self BI**: 사용자가 직접 데이터 업로드 후 개인 보고서 즉시 작성
- **사용자 무제한 라이선스**: 사용자 확장에 따른 추가 비용 미발생
- **단일지원체계**: 제조·공급·구축·지원 One-stop

---

## 4. 기술 스택

- **웹**: 100% HTML5, Java 기반
- **OLAP 엔진**: 특허 보유 고성능 OLAP 엔진
- **분석 엔진**: HDFS, Hive, Spark SQL, Mahout 빅데이터 연동
- **배포 방식**: On-premise / SaaS (클라우드)

## 7. 제안서 작성 시 강조 포인트

> LLM이 제안서를 생성할 때 이 섹션을 우선 참고합니다.

- **국내 최초 GS인증 1등급 BI 솔루션** — 검증된 품질로 공공 조달 즉시 적용 가능
- **외산 솔루션(SAP·MicroStrategy) Win-back 실적** — 국내 환경에 최적화된 기술력
- **AI 기반 생성형 보고서** — LLM으로 정기 보고서 자동 생성, 보고서 작성 업무 자동화
- **사용자 무제한 라이선스** — TCO 절감 효과 수치로 제시
- **200여 개 공공·금융 기관 납품 실적** — 레퍼런스 기반 신뢰성 강조
- **데이터버틀러 AI 기능 탑재** — 메타데이터 기반 자동 분석 추천, 진단 자동화

# 와이즈AIOps (WiseAIOps)

> **분류:** AI 개발 플랫폼 (MLOps)  
> **최종 수정일:** 2026-05-20

---

## 1. 제품 개요

데이터 전처리, 모델 개발·학습, 평가, 모니터링, 배포까지  
**AI 개발 전 단계를 하나의 환경에서 지원**하는 엔드투엔드 AI 플랫폼.

---

## 2. 주요 기능

| 기능 | 설명 |
|------|------|
| 데이터 전처리 | 다양한 데이터 소스 연결, 정제·변환 파이프라인 구성 |
| 모델 개발·학습 | 주피터 환경 기반 모델 개발, 분산 학습 지원 |
| 모델 평가 | 성능 지표 자동 계산 및 실험 비교 관리 |
| 모델 모니터링 | 배포 후 모델 성능 드리프트 감지 및 알림 |
| 모델 배포 | API 형태 자동 배포, 버전 관리 |
| 파이프라인 자동화 | 재학습·배포 파이프라인 스케줄링 |

---

## 3. 도입 효과

- AI 개발부터 운영까지 일원화된 관리 환경 제공
- 모델 재현성 보장 및 버전 추적
- 망분리 환경 온프레미스 구축 지원

---

## 4. 제안서 작성 시 강조 포인트

- **엔드투엔드 AI 플랫폼** — 단계별 툴 난립 없이 단일 환경에서 전 과정 관리
- **공공기관 망분리 환경 지원** — 보안 요구사항 충족
- **AI 모델 거버넌스** — 모델 버전·이력·성능 추적으로 설명 가능한 AI 구현
- **WiseProphet 연계** — 비전문가도 활용 가능한 저코드 AI 개발 환경 통합 제공

# 데이터버틀러 (Data Butler)

> **데이터버틀러**는 위세아이텍의 AI 기능 전체를 통칭하는 **브랜딩 명칭**입니다.  
> 특정 단일 제품이 아니라, 각 제품에 탑재된 AI 기능들의 집합을 의미합니다.

---

## 개요

데이터버틀러는 위세아이텍의 Data Management 제품군(WiseDQ, WiseMeta 등)에  
내장된 AI 자동화 기능들을 통칭하는 브랜드입니다.

**목표:** 데이터 관리의 반복적·수작업 업무를 AI로 자동화하여 운영 효율을 극대화

---

## 사업 영역별 데이터버틀러 AI 기능

| 사업 영역 | 파일 | 탑재 제품 | 핵심 AI 기능 |
|-----------|------|-----------|-------------|
| 메타데이터 관리 | [META_AI기능.md](./META_AI기능.md) | WiseMeta | 표준용어 자동 추천 |
| 데이터 품질 관리 | [DQ_AI기능.md](./DQ_AI기능.md) | WiseDQ | 검증룰 자동 매핑, 간편 진단 |

---

## 데이터버틀러 전체 기능 요약

### META 영역 (WiseMeta)

| 기능 | 구현 방향 | 효과 |
|------|-----------|------|
| 표준용어 추천 고도화 | 룰 기반 + 의미 기반 + ML 기반 | 사용자 작업 부담 감소, 작업 시간 단축 |

### DQ 영역 (WiseDQ)

| 기능 | 구현 방향 | 효과 |
|------|-----------|------|
| 검증룰 자동 매핑 | 룰 기반 + ML 기반 | 품질 진단 셋업 시간 단축 |
| 업무규칙 후보 자동 생성 | ML 기반 | 업무규칙 작성 부담 감소 |
| 자동 제외 룰 생성 | 룰 기반 | 진단 효율성 향상 |
| 간편 진단 | 룰 기반 + ML 기반 연계 | 스키마 선택만으로 초기 진단 환경 구성 |

---

## 제안서 활용 방법

데이터버틀러 AI 기능은 다음과 같이 제안서에서 활용합니다:

1. **AI 자동화 강조**: "데이터버틀러 AI 기능으로 기존 수작업 대비 XX% 효율화"
2. **지능형 플랫폼 포지셔닝**: 단순 솔루션이 아닌 AI 내장 스마트 플랫폼으로 차별화
3. **지속 고도화 로드맵**: ML 기반 기능이 지속 발전 중임을 경쟁력으로 제시

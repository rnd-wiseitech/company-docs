# 데이터버틀러 — DQ 영역 AI 기능

> **탑재 제품:** [WiseDQ](../products/와이즈DQ.md)  
> **최종 수정일:** 2026-05-20

---

## 개요

WiseDQ에 탑재된 데이터버틀러 AI 기능입니다.  
데이터 품질 진단의 준비·설정·실행 단계에서 수작업을 AI로 자동화하여  
**진단 셋업 시간 단축과 일관된 진단 기준 적용**을 지원합니다.

---

## AI 기능 목록

### 1. 검증룰 자동 매핑 기능 고도화

| 항목 | 내용 |
|------|------|
| **기능 설명** | 수집된 DB 카탈로그 및 컬럼 메타데이터를 기반으로 컬럼 특성에 적합한 검증룰 후보를 자동 추천 |
| **구현 방향** | 룰 기반 + ML 기반 |
| **주요 효과** | 품질 진단 준비 과정의 수작업을 줄이고 초기 진단 셋업 시간 단축 지원 |

---

### 2. 업무규칙 후보 자동 생성

| 항목 | 내용 |
|------|------|
| **기능 설명** | 컬럼 정보를 기반으로 품질 진단에 활용 가능한 업무규칙 후보를 자동 생성·추천 |
| **구현 방향** | ML 기반 |
| **주요 효과** | 업무규칙 작성 부담을 줄이고 진단 기준 수립 지원 |

---

### 3. 자동 제외 룰 생성

| 항목 | 내용 |
|------|------|
| **기능 설명** | 컬럼 정보를 기준으로 테이블 및 컬럼 진단 제외 대상 후보를 자동 선별 |
| **구현 방향** | 룰 기반 |
| **주요 효과** | 불필요한 진단 대상을 줄여 진단 효율성 향상 |

---

### 4. 간편 진단

| 항목 | 내용 |
|------|------|
| **기능 설명** | 검증룰 자동 매핑 + 업무규칙 후보 추천 + 자동 제외 대상 선별을 통합하여 **스키마 선택만으로** 초기 진단 환경 구성 지원 |
| **구현 방향** | 룰 기반 + ML 기반 연계 |
| **주요 효과** | 진단 셋업 시간 대폭 단축, 일관된 진단 기준 적용 |

---

## 전체 기능 흐름

```
[스키마 선택]
      ↓
[검증룰 자동 매핑] ──── DB 카탈로그 + 컬럼 메타데이터 분석 (ML)
      ↓
[업무규칙 후보 자동 생성] ── 컬럼 특성 기반 규칙 추천 (ML)
      ↓
[자동 제외 룰 생성] ──── 진단 불필요 대상 자동 선별 (룰 기반)
      ↓
[간편 진단 실행] ──────── 3단계 통합 자동화 → 즉시 품질 진단 시작
```

---

## 제안서 활용 포인트

- **진단 셋업 자동화**: 기존 수작업 대비 셋업 시간 대폭 단축 → 수치로 제시
- **간편 진단**: 스키마 선택 한 번으로 초기 품질 진단 환경 즉시 구성
- **ML 기반 지속 고도화**: 진단 이력이 쌓일수록 추천 정확도 향상
- **공공 데이터 품질법 대응**: 일관된 기준으로 법적 의무 진단 체계 구축

## 개요

WiseMeta에 탑재된 데이터버틀러 AI 기능입니다.  
메타데이터 관리 업무 중 반복적·수작업 영역을 AI로 자동화합니다.

---

## AI 기능 목록

### 표준용어 추천 기능 고도화

| 항목 | 내용 |
|------|------|
| **기능 설명** | 입력된 논리명에 부합하는 표준 물리명을 자동 추천하는 기능 |
| **구현 방향** | 룰 기반 + 의미 기반 + ML 기반 (컬럼명 패턴 분석, 도메인 유형 분류 등) |
| **주요 효과** | 표준 물리명 추천으로 사용자 작업 부담 감소 및 작업 시간 단축 |

#### 동작 원리

```
[사용자 입력] 논리명 (예: "고객식별번호")
        ↓
[룰 기반] 기 정의된 용어 매핑 규칙 적용
        ↓
[의미 기반] 임베딩 유사도로 의미 근접 용어 탐색
        ↓
[ML 기반] 컬럼명 패턴 분석 + 도메인 유형 분류 모델 적용
        ↓
[출력] 표준 물리명 후보 목록 + 신뢰도 점수
```

---

## 제안서 활용 포인트

- **메타데이터 구축 자동화**: 표준용어 작업 인력·시간 절감 수치로 제시
- **3중 검증 체계**: 룰·의미·ML 기반 3단계 추천으로 높은 정확도 보장
- **데이터 표준화 가속**: 기관 전체 데이터 표준 체계 구축 기간 단축
- **지속 학습**: ML 모델이 사용 이력으로 지속 고도화되는 자기 개선 시스템

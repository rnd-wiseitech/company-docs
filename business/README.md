## 작성 규칙

- 파일명: 사업 영역명을 한글로 (예: `AI데이터플랫폼.md`, `데이터관리.md`)
- 수행 실적은 계약 완료 후 기재, 진행 중 사업은 `(진행중)` 표기
- 새 사업 영역 추가 시 `_양식.md` 복사 후 이 README 목록에도 추가

# 주요 사업 영역

와이즈아이텍의 사업 영역별 역량과 수행 실적 문서 목록입니다.  
제안서 작성 시 해당 사업 영역의 문서를 LLM 컨텍스트로 활용합니다.

---

## 사업 영역 목록

| 파일 | 사업 영역 | 주요 고객 |
|------|-----------|-----------|
| [_양식.md](./_양식.md) | 작성 양식 샘플 | — |

> 새 사업 영역 문서를 추가할 때는 `_양식.md`를 복사해 `영역명.md`로 저장하세요.

---

## 작성 규칙

- 파일명: 사업 영역명을 한글로 (예: `공공SI.md`, `AI솔루션.md`, `정보보안.md`)
- 수행 실적은 계약 완료 후 기재, 진행 중인 사업은 `(진행중)` 표기
- 금액 등 민감 정보는 대략적인 규모(억 원 단위)로 표기

---

## 폴더 구조 예시

```
business/
├── README.md          ← 이 파일 (사업 영역 인덱스)
├── _양식.md            ← 새 사업 영역 작성 시 복사할 양식
├── 공공SI.md
├── AI솔루션.md
├── 정보보안.md
├── 빅데이터플랫폼.md
└── 디지털전환컨설팅.md
```

# 데이터 관리 사업 (Data Management)

> **최종 수정일:** 2026-05-20

---

## 1. 사업 영역 개요

위세아이텍의 데이터 관리 솔루션 사업 영역입니다.  
데이터 품질 관리, 메타데이터 표준화, 시스템 영향도 분석, 소프트웨어 보안을 통합 제공하여  
**데이터 거버넌스 체계 수립부터 운영까지 전 과정을 지원**합니다.

**핵심 가치:** AI 기반 자동화로 데이터 관리 인력·비용 절감, 데이터 신뢰성 확보

---

## 3. 핵심 역량

- **AI 기반 데이터 품질 자동화** — 데이터버틀러 AI 기능으로 진단 셋업 자동화
- **공공 데이터 품질 관리법 대응 전문성** — 법적 요구사항 충족 체계 수립
- **표준용어 AI 추천** — ML 기반 논리명→물리명 자동 매핑으로 표준화 작업 효율화
- **SW 보안 내재화** — 개발 단계 보안 취약점 자동 탐지 (Shift Left Security)

## 7. 제안서 작성 시 강조 포인트

- **공공 데이터 품질 관리법 의무 대응** — 법적 요구사항 충족 솔루션
- **AI 기반 진단 자동화** — 데이터버틀러로 기존 수작업 대비 효율화 수치 제시
- **메타·품질 통합 거버넌스** — 단일 벤더로 데이터 거버넌스 전 범위 커버
- **SW 보안 규정 대응** — 행안부 SW 보안 취약점 진단 요구사항 충족
